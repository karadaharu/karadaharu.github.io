PHP配列

array key valueがある

$array[3] = 'あ';
$array[2] = 'い';
$array[1] = 'う';
$array[0] = 'え';

foreach($array as $key => $value){
  print $key . ":" . $value . <br />";
}

↓ 3:あ 2:い 1:う 0:え

foreachではkeyの順番には出てこない。 単純にarrayに突っ込んだ順。
