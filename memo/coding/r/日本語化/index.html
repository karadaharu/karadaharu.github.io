日本語化

日本語化に関するもろもろの知識まとめ

plot()等の作図関数はQuartzという描画エンジンを使っている ここで使われるフォントのセットを定義するのがquartzFont フォントブックのPostScript 名でfont familyを定義できる．

latticeの設定は基本的にlattice.options()のテーマを変えることで行なう．

trellis.par.set:latticeのグラフィックパラメータをカスタマイズする ここにはfont familyの設定はない？

lattice.options:その他の設定 trellis.device:グラフィックパラメータの初期設定

xyplot(Murder ~ Population | state.region, data = states,
  groups = state.name,
  panel = function(x, y, subscripts, groups) {
    ltext(x = x, y = y, labels = groups[subscripts], cex=1,
      fontfamily = "HersheySans")
      })

panel:?

simpleKey(fontfamily=“hoge”):simple interface to generate a list appropriate for draw.key

結果だけまとめ

Quartzのフォントファミリーに日本語が表示できるものを追加

quartzFonts( hira = c("HiraMaruProN-W4","HiraKakuProN-W3","HiraMinPro-W3","HiraKakuPro-W6"))

各描画関数でfamilyを指定

pl <- xyplot(0 ~ 0 | "Stripあ Text",main="い",par.settings = list( grid.pars = list(fontfamily = 'hira')))

デフォルトを変更する場合は

trellis.par.set(grid.pars = list(fontfamily = "hira"))
