pythonからsqliteを使う

#pythonからsqliteを使う

##SQLiteって？ structured query language

テーブルとフィールドがある フィールド：行と列が交わった場所

http://www.dbonline.jp/sqlite/table/index2.html データベース内にテーブルを作成した場合、SQLiteの特別なテーブル「sqlite_master」テーブル内に実行されたCREATE TABLE文が格納されています。

sqlという言語があるので、pythonかどうかは関係なく検索したほうがいい

公式ドキュメント　分かりやすい http://docs.python.org/2/library/sqlite3.html

テーブルの名前を出す select * from sqlite_master;

r = c.fetchone()

tuple型:変更できないリスト型

show tables:すべてのテーブルの名前を表示する

Pythonによる基本的な使い方 http://www.gesource.jp/programming/python/code/0013.html

返り値の確認

  • a = c.execute(hoge)で，aはsqlite3.cursorクラス
  • r = a.fetcheone()で中がとれる．（r in aではとれないことも)

WHEREの使い方

id = 10 cur.execute(“select * from data where id = ?“, (id,))

ORDER BY でソートするとき

id = 10 cur.execute(“select * from data where id = ? ORDER BY seq”, (id,))#OK cur.execute(“select * from data ORDER BY seq where id = ?“, (id,))#BAD

execute()の返り値

a = cur.execute(“hoge”) のaはポインタ的なやつっぽい もう一回execute()すると，値が変わってしまう
