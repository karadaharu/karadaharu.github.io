代入

データフレームのどこかに値を入れるときは、そのアクセスの仕方が、コピーを返すのか、ビューを返すのかをちゃんと区別しないといけない。基本的にチェーンアクセスをすると、複数回メソッドが呼ばれて、その最初のメソッドはget系でコピーになるのでダメ

代入できる例

dfmi.loc[:,('one','second')] = value
# becomes
dfmi.loc.__setitem__((slice(None), ('one', 'second')), value)

代入できない例(できるけどwarningが出る場合もある)

dfmi['one']['second'] = value
# becomes
dfmi.__getitem__('one').__setitem__('second', value)

1つのセルに入れるときの最も良いやり方

dfmi.set_value(ind, col, val)
