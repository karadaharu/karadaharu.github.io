モジュール

Pythonではフォルダに「init.py」があれば、そのフォルダを「パッケージ」として扱えるようになっています。

モジュールオブジェクトはシングルトンになっているので複数回importしても大丈夫

パッケージはPython のモジュール名前空間を “ドット付きモジュール名” を使って構造化する手段

AttributeError: ‘module’ object has no attribute のエラーの原因と対処法

循環参照になってしまっている→必要な関数の中でだけimportする

基本モジュールと名前がかぶっている→ 絶対import機能を使う

_document_frequency _make_int_array _check_stop_list

名前空間

from hoge import *

でロードされるのは、基本アンダースコアから始まらないもののみ。__all__変数が定義されていると、アンダースコアから始まるのもロードされる。 http://stackoverflow.com/questions/2360724/what-exactly-does-import-import
