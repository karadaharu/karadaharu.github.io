pythonと文字コード

pythonと文字コード

以下はpython2.xにおける話。 python3.xではstr型はなくなり、すべてユニコード型

基本

  • 下記ソースコードの文字コードを指定している

    #-*- coding:utf-8 -*-

  • ユニコードは文字コードの一種ではない

  • unicode 型は「文字コード」にとらわれない型で、CPU やメモリ上で用いられる

  • Python の str 型は「文字コード」に則ったバイト列で、出入力時にはこれを用いる

基本の詳細

  • unicode型はprintして標準出力に出力される際に、指定された文字コードのstr型に変換されて出力される
  • ASCIIに定義されている文字列は特権的に柔軟に使える。(ASCII:128文字のみ。)
