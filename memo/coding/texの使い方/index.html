コンパイルの基本

ファイルtest.texに対して、

$ uplatex test
$ dvipdfmx test

uplatexでdviファイルを作成 dvipdfmxでdviファイルからpdfを作成 dviファイルとは、texの出力ファイルフォーマット。

または一発で

ptex2pdf -u -l test

bibtexを使う場合

$ uplatex test
$ pbibtex test
$ uplatex test
$ dvipdfmx test

でできる
