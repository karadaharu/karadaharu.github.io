##名前空間

ifndefとendif

擬似命令:#ifndef,#endif. プリプロセッサ(コンパイル前に実行される)によって処理される命令．条件コンパイルができる． #ifndef INCLUDEDSample_h #define INCLUDEDSample_h #endif INCLUDED_Sample_h_を記号定数と呼ぶ．#defineで一度記号定数が定義されていたら，#ifndefから#endifまではコンパイルされない．これによって，ひとつのプログラムが何度もインクルードされなくなる． #pragma once でも同じようなことができるらしい
