opencv

OpenCV

基本

画像を保持するクラス・構造体

  • CvMat:行列を保持する構造体（古い）
  • CvArr:実体はvoid。関数の引数はこれになっていて、CvMatとIplImageを同列に扱うためにvoidポインタになっていて、内部で判断してくれる。InputArrayもたぶん同じ感じ。
  • cv::Mat:C++版で画像も行列も一様に扱うクラス。メインはこれ。クラスなので色々な処理をするメソッドがついてる。

Mat型変数には画素の型がたくさんある。

二値化

cvThreshold(const CvArr *src, CvArr *dst, double threshold, double max_value, int threshhold_type);
