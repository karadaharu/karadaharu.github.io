C言語

C言語

C99以降では、変数を配列の大きさとして使える。

文字列

‘A’はchar型、”ABC”は’A’のアドレス

#include <string.h>
int strcmp(const char *s1, const char *s2);

s1>s2のとき正、s2>s1のとき負。 大小関係は文字コードによる。

アスキーコードの場合、大文字のほうが先なので、

int result = strcmp("BBB","aaa") // result < 0

複数ファイルの扱い

ヘッダーファイルを重複して読み込むとエラーになる。以下のように書くことで二回目以降の読み込みをしないようにできる。

#ifndef _SAMPLE
#define _SAMPLE

// 中身

#endif
