SSLとかTSLとか

SSLとかTLSとか

SSL:Secure Sockets Layer　ネットスケープが開発した技術の名前 TLS:Transport Layer Security　標準化された名前

https:SSLまたはTLSで暗号化している

SSL通信の流れ

  1. クライアントがサーバーにhttpsでアクセス
  2. サーバーは証明書を送付
  3. クライアントが乱数を生成、証明書にある公開鍵で暗号化、送付
  4. 以降は互いに乱数から生成した共通鍵で暗号化して通信

証明書が正しいと判断できるのは、認証局(CA)が発行した公開鍵証明書がブラウザインストール時に入るから。
