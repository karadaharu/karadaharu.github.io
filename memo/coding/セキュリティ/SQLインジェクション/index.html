SQLインジェクション

SQLインジェクション

$pdo = new PDO('sqlite:/path/db/users.db');
$pdo->query("SELECT name FROM users WHERE id = " . $_GET['id']); // <-- ダメ、ゼッタイ！

もしも、http://domain.com/?id=1%3BDELETE+FROM+usersみたいなURLをたたくと、SQL文はSELECT name FROM users WHERE id = 1; DELETE FROM usersが実行されてしまう。こう書く。

<?php
$pdo = new PDO('sqlite:/path/db/users.db');
$stmt = $pdo->prepare('SELECT name FROM users WHERE id = :id');
$stmt->bindParam(':id', $_GET['id'], PDO::PARAM_INT); // <-- PDOが自動的にエスケープ処理をする
$stmt->execute();
