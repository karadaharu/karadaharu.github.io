postgresが停止できなかった話

postgresが停止できなかった話

pg_ctl -m immediate stop で停止ができない！

pg_ctl status または ps alx | grep postgres でpostgresの親プロセスのPIDを探して kill -9 PID をしても、また復活してしまう！

ps alx | grep postgres でpostgresの親プロセスの親プロセスが /sbin/launchd であることを発見

launchdが一番最初にシェルを起動させたりもろもろ設定するプロセスだと知る

launchdで何が起動するのか指定しているのは？ launchctl list で指定されているもの一覧が見れることが分かる

grep postgres でpostgres関連のが見つけれられるはず。案の定、 homebrew.mxcl.postgresql が見つかる。

homebrew.mxcl.postgresql でぐぐってみたところ、 ここでhomebrew でサービスを指定していることが分かる。 brew services restart hogeというサブコマンドがあることがわかり、stopもできそうだけど、とあたりをつけて brew servicesで検索したところ、service とbrew とlaunchdを一緒に解説している記事を発見

一件落着。
