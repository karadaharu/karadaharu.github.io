ポートフォワード

ポートフォワード

特定のネットワーク内からしか見れないウェブページを見るためには、 そのネットワーク内にある端末のポートを使って プロキシサーバーに使ってアクセスします。 http://rcmdnk.github.io/blog/2014/09/04/computer-remote/

プロキシサーバー

proxy:代理 ユーザーのコンピュータの代わりに目的のコンテンツを取ってくる機能を提供するサーバーのこと。

プロキシサーバーのメリット

  • キャッシング
  • フィルタリング
  • 通信経路の変更
  • 匿名性確保

ポートフォワーディング

通常は各アプリケーションがそれぞれのポートを使って通信しているところを、別な通信経路を用いて特定のポートと通信すること。

ex)外部からアクセスできない会社のメールサーバーをSSHを経由して利用する。

Socks

内部ネットワークからインターネットに接続するためのプロトコルの一種。 HTTPなどよりも低いセッション層のプロトコルであるため、全てのアプリケーション層のプロトコルに対応したプロキシサーバーとして使える。（キャッシング機能はない）

実践

$ ssh -N -f -D 10080 user@example.com # あとでプロセスをkillする。-fを外せばバックグラウンドにいかない

-N:リモートコマンドを実行しません。 -D:アプリケーションレベルの”動的な”ポート転送を指定します。

Firefoxで ‘>環境設定>詳細>手動でプロキシを設定する

SOCKSホスト:localhost ポート:10080 に設定 すればOK

それ以外のブラウザで設定するとパソコン全体がそうなってしまう。
