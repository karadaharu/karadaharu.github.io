ドメインとサーバー

ドメインとサーバー

DNSサーバー(ネームサーバー)

DNSサーバー：ドメイン名→IPアドレスに変換する →DNS情報を変更しないといけない

レジストラがドメインを管理していて、どのネームサーバーを見に行けばいいのかを登録してある。 ドメインにおいて、どのネームサーバーかが分かり ネームサーバーにおいて、どのIPアドレスかが分かる

レジストリとレジストラ

  • レジストリ:トップレベルドメイン名ごとにひとつの管理組織。
  • レジストラ:ドメイン名の登録申請をうけつける組織

DNS

DNSレコードとは？

ドメインに関する情報（ドメインのDNSサーバ名、管理者のメールアドレス等）

A:ホストのIPアドレス CNAME:ホスト名のエイリアス AAAA:IPアドレス(IPv6)

DNSゾーンとは？

ネームサーバーがドメインを管理する範囲のこと

なぜひとつのIPアドレスで、複数のドメインに対応できるのか？

サーバーを仮想化している（VPSなど） 安いレンタルサーバーは、ひとつのサーバーを共有して、マルチドメイン方式で複数のドメインを運用している

gTLD

generic TOP LEVEL DOMAIN 地理的に関係ないもの(.comなど)
