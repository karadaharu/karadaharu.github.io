シェルスクリプトの書き方

シェルスクリプトの書き方

変数の宣言

string="aaaa"
echo ${string}

変数の=の両側にはスペースを入れてはいけない。入れると、変数名がコマンドとして受け取られる 変数の中身を使いたいときは,${}で囲む。そうしないとコマンドや、文字列として受け取られる

forループ

files=*txt
for f in files
do
	echo ${f}
done

if文

if test hoge
else
fi

対話的なコマンドの実行

#!/bin/sh
expect -c "
set timeout 400
spawn ftp ladsweb.nascom.nasa.gov
expect \"Name (ladsweb.nascom.nasa.gov:kazumasa):\"
send \"anonymous\n\"
expect \"Password:\"
send \"catter121@gmail.com\n\"
expect \"ftp>\"
send \"cd /orders/\n\"
expect \"ftp>\"
send \"binary\"
expect \"ftp>\"
"
  • respawnだとシェルのコマンドが打てる
  • 最後は\n入れないとエンターにならない

ecpect はTclの文法をわかっていないと戸惑うらしい。 http://root-mode2.blogspot.jp/2010/10/expect-spawn.html

テキストファイルを一行ずつ読み込む

while read line
do
	echo $line
done < test.txt
