Webアプリケーション・サーバーとCGIは違う CGI:あるファイルにアクセスすると、そのファイルが実行されて何かが返ってくる Webアプリケーション・サーバー:サーバーが起動するときにスクリプトファイルを読み込み、サーバーへのリクエストに対して、何かを返す

フレームワーク: テンプレートエンジンなどの部品を集めて利用しやすくした フレームワークの種類のわけ方: リクエストハンドリングの方法:あるリクエストに対応するのが、関数かクラスか テンプレートエンジンの記法:いろいろtumblrとwordpressの違い、みたいなことだと思う フレームワークごとに書き方が違うと、移行するときに全部コードを書き直す必要がある→WSGI(標準化)
