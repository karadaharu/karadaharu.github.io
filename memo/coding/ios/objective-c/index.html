objective-c

Objective-C

クラス

/*宣言*/
@interface MyClass : NSObject {
	NSString *testString // メンバ変数
}
@end
/*実装*/
@implementation MyClass: NSObject
@end

基本はプロパティには直接アクセスしないで、アクセサメソッドで扱う。

クラスのインスタンス生成

[[UIButton alloc] init]

alloc:メモリ割り当て init:イニシャライザ　initWithXxxで変数を追加してオーバーライド

プロパティ属性

  1. アクセス制御
  2. 所有属性
  3. アトミックの指定
  4. アクセサ名の指定

ARC、メモリ管理をちゃんとやるのに必要

ふつうのUIButton UIButton* contactButton = [UIButton buttonWithType:UIButtonTypeSystem];

@がつくものはコンパイラディレクティブ：特定の命令を前処理するように指示するもの

strong:自分がオーナーシップをもつ weak:参照先が解放されたらnilがはいる
