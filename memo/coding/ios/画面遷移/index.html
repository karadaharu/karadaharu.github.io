画面遷移

画面遷移

画面に関わるクラス

  • UIStorybord
  • UIViewController
  • UINavigationController
  • UITabBarController

基本

  • ストーリーボード上の画面1枚とController1つが対応している
  • Segue:シーン（ストーリーボード上の1画面）とシーンをくっつけるもの

Segue

Segueには4種類ある

  1. Show:現在のViewControllerの1階層奥に押し出す。アニメーション：右から左へスライド
  2. Show Detail:現在のViewControllerを置き換える。iPadの画面分割の右半分など。
  3. Present Modally:現在のViewControllerの上に重ねる。
  4. Present As Popover:新しいViewControllerをポップオーバー。iPad用

ストーリーボード上で画面遷移を設定する

Segueを使う。使い方：シーンの上左のアイコンを右クリック、manualから次のシーンにつなぐ。modalを選択。Segue(とぎれなく続く)ができる

別のストーリーボードのシーンへ移動する

タブを使っていないときでストーリーボードのイニシャルシーンへ移動

let storyboard: UIStoryboard = UIStoryboard(name: "HomeViewController", bundle: nil)
let next: HomeViewController = storyboard.instantiateInitialViewController() as HomeViewController
presentViewController(next, animated: true, completion: nil)

コントローラー内で完結したシーンへ移動

self.presentViewController(mySecondViewController, animated: true, completion: {println("遷移!")})

self.view.addSubview(var1) でvar1を画面に追加できるが、ストーリーボードを使うときは別。
