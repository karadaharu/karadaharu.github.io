select

  • 探している値があるかどうか確認する:SELECT * FROM t1 WHERE EXISTS(SELECT * FROM t2 WHERE t1.id = t2.t1_id);
  • 複数の値について探す:SELECT * FROM temp WHERE test IN(1,3,5,10);
  • 他のテーブルの値について探す:SELECT * FROM temp WHERE test IN(select id from list);
